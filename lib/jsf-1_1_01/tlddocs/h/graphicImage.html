Tag graphicImage

Renders an HTML "img" element. Render the clientId as the value of the "id" attribute. Render the value of the component as the value of the "src" attribute, after passing it to the getResourceUR() method of the ViewHandler for this application, and passing the result through the encodeResourceURL() method of theExternalContext. If present, render the value of the alt attribute as the value of the "alt" attribute. If the "styleClass" attribute is specified, render its value as the value of the "class" attribute.
